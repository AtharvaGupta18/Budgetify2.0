## ✨ About Budgetify

**Budgetify** is a modern **money management and expense tracking application** that helps users become more aware of their spending habits.

Track your **daily expenses**, record your **income**, and maintain a clear overview of your finances in one place.

Budgetify makes managing money simple, organized, and consistent.

# 🚀 Features

### 💰 Expense Tracking

- Organize spending records
- Maintain financial awareness

### 💵 Income Tracking
- Record income entries
- Track earnings over time

### 📅 Daily Money Logging
- Simple day-to-day finance management
- Quick entry experience

### 🔍 Spending Awareness
- Understand where money goes
- Encourage better financial decisions

### ☁️ Cloud Sync
- Data stored securely with Firebase

# ⚙ Installation

```bash
git clone https://github.com/AtharvaGupta18/Budgetify2.0.git

cd Budgetify2.0

npx expo install

npx expo start
```
